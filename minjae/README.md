## Command

0. install express
```
npm i express
```

1. init npm
```
npm init -y
```

2. start node server
```
node index.js
```

3. install nodemon which is used for auto-restart server when code is modified
```
npm i -D nodemon
```

4. write command for start and nodemon in package.json

